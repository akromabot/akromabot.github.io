Commandes Description
$gif Envoie une photo de profil animée.
$png envoie une photo de profil non animée.
$banner envoie une bannière.
$avatar envoie l'avatar de la personne mentionée.
$clear supprime des messages.
$banlist Permet de voir le nombre de ban.
$infos Donne les informations du robot.
$invite Envoie le lien dinvitation du robot.
$ping Affiche le ping actuel du robot.
$uptime Affiche depuis quand le robot est allumé.
$serverinfo Affiche les informations du serveur.
$roleinfo Affiche les informations du rôle mentioné.
